# rancher安装nfs-client-provisioner动态提供nfs存储

# 第一步现在先添加应用商店：

商店1地址：

http://mirror.azure.cn/kubernetes/charts-incubator/ 商店2地址：

http://mirror.azure.cn/kubernetes/charts/ 路径如下：全局-工具-商店设置-添加应用商店

# 第二步：启用应用商店，搜索 nfs-client-provisioner 进行安装，

添加应答编辑框输入，nfs-server的ip地址与共享路径

nfs.server 192.168.101.12 nfs.path /nfs

点击启动即可。
