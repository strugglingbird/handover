      # rancher中nginx-ingress的配置获取真实IP地址

      解决nginx-ingress无法从4层负载均衡器通过protocol协议转发的请求中获取客户端的真实IP地址问题

      # 配置rancher

      • 切换到【System】项目中

      • 进入【配置映射】中

      • 找到命名空间ingress-nginx中的nginx-configuration配置

      • 打开配置编辑

      • 添加

        键 值
        compute-full-forwarded-for true
        use-forwarded-headers true
        use-proxy-protocol true

      # 配置负载均衡服务器nginx

      • 登陆服务器

        ssh root@192.168.1.71
