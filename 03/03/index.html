# rancher升级kubernates版本

# 下载最新版本RKE

  • 进入rke的下载页 (opens new window)进行下载

  • 下载完成后将下载的文件命名成rke并替换掉原来的rke执行文件

# 升级版本

  • 查看rke支持kubernates的版本号

    [root@lb rke-install]# ./rke config --list-version --all
v1.19.10-rancher1-1
v1.17.17-rancher2-3
v1.18.18-rancher1-2
v1.20.6-rancher1-1
